# Specification: フロントエンド開発 - 効率的フロンティア画面の実装

## 概要
Next.js 14を使用して、投資シミュレーションアプリケーションの基盤となるダッシュボードレイアウトを構築し、最初の主要機能として「効率的フロンティア」の計算・視覚化画面を実装する。

## 機能要件
1.  **ダッシュボードレイアウト:**
    - 常設のサイドバーを配置し、今後の機能（モンテカルロ等）へのナビゲーションを提供。
    - レスポンシブ対応のメインコンテンツエリア。

2.  **効率的フロンティア計算フォーム:**
    - API (`/api/assets`) から資産一覧を取得し、計算に含める資産（チェックボックス形式等）を選択可能にする。
    - 計算ポイント数などのパラメータ設定。

3.  **インタラクティブ・視覚化 (Plotly.js):**
    - リスクを横軸、期待リターンを縦軸とした散布図（フロンティア曲線）の描画。
    - **ホバー機能:** 各点のリターン・リスク値を詳細表示。
    - **ズーム・パン:** グラフ範囲の自由な操作。
    - **ポイント選択:** グラフ上の点をクリックすると、後述のテーブルにその配分内容が連動。

4.  **資産配分詳細テーブル:**
    - グラフで選択された、または特定の計算点における具体的な「銘柄名：比率」を一覧表示。

## 非機能要件
- **ユーザー体験:** shadcn/uiを使用し、Material Designの原則に基づいたモダンで清潔感のあるUIを実現。
- **パフォーマンス:** 状態管理にZustand、データ取得にTanStack Queryを使用し、スムーズな画面遷移とデータ連携を実現。

## 受入条件
- 資産を選択して「計算」を実行し、正しいフロンティア曲線が表示されること。
- グラフ上の点をクリックした際、テーブルの資産配分比率が正しく更新されること。
- サイドバーによるナビゲーションが機能していること。
