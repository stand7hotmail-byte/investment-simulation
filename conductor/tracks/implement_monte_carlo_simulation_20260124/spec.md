# Specification: モンテカルロシミュレーションの実装

## 概要
ユーザーのポートフォリオ（資産配分）と資産マスタデータに基づき、将来の資産形成を確率的に予測するモンテカルロシミュレーション機能を実装する。

## 機能要件
1.  **ポートフォリオ特性の算出:**
    - ユーザーが作成したポートフォリオの資産配分と、各資産の期待リターン・ボラティリティ・相関係数から、ポートフォリオ全体の期待リターンとボラティリティを算出する。

2.  **シミュレーションエンジン:**
    - **手法:** 幾何ブラウン運動（対数正規分布）を用いた離散時間シミュレーション。
    - **入力パラメータ:**
        - 初期投資額 (Initial Investment)
        - 毎月の積み立て額 (Monthly Contribution)
        - シミュレーション期間 (Years)
        - 任意のタイミングでの追加投資 (Extra Investments)
        - 試行回数 (例: 10,000回)
    - **計算ロジック:** 年次または月次ステップで将来価値を算出し、試行回数分繰り返す。

3.  **APIエンドポイント (`POST /api/simulate/monte-carlo`):**
    - ポートフォリオIDまたは資産配分を受け取り、計算結果を返す。

4.  **出力データ:**
    - **統計値:** 最終資産額の分布（10%, 25%, 50%, 75%, 90%タイル）。
    - **推移データ:** 期間中の資産推移（中央値、および上位・下位の境界ライン）。
    - **確率指標:** 元本割れ確率、ユーザー指定の目標金額への到達確率。

## 非機能要件
- **計算パフォーマンス:** 10,000回の試行を1秒以内に完了させる。
- **精度:** 統計的に妥当な乱数生成器（NumPy等）を使用する。

## 受入条件
- ポートフォリオを指定してシミュレーションを実行し、統計データとチャートデータが正しく返ってくること。
- 追加投資（任意タイミング）がシミュレーション結果に正しく反映されること。
- ユニットテストで計算結果の妥当性が検証されていること。

## 範囲外 (Out of Scope)
- 税金や手数料の精緻な計算（今回は考慮しない）。
- 期間中のリバランスの自動実行。
