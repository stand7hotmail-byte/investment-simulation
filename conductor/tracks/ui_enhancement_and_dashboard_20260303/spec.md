# Specification: クリーン・ミニマルなUI改善とダッシュボードの導入 (UI_Enhancement_and_Dashboard)

## 1. 概要 (Overview)
現在の投資シミュレーションアプリのUIを、よりモダンで洗練された「クリーン・ミニマル」なデザインに刷新する。情報の整理を優先し、カードベースのレイアウトや折りたたみ可能なサイドバー、そして全体を概観できるダッシュボード画面を新規導入することで、ユーザー体験（UX）の質を向上させる。装飾的なアニメーションは排除し、機能的で美しい実用的なツールを目指す。

## 2. 機能要件 (Functional Requirements)

### 2.1 全体的なデザイン・テーマの刷新
- **カラーパレット:** 白 (#FFFFFF) および薄いグレー (#F3F4F6, #F9FAFB) を基調とした背景。アクセントカラーには落ち着いた青または緑を採用。
- **カードレイアウト:** 各セクション（グラフ、テーブル、フォーム等）を、独立した「カード」（背景白、微細なドロップシャドウまたは境界線）内に配置し、情報の区切りを明確にする。

### 2.2 ナビゲーションの改善 (Sidebar)
- **折りたたみ機能:** サイドバーをアイコンのみの細い状態に折りたたみ、コンテンツエリアを広く確保できる機能を実装。
- **アイコンとラベル:** Lucide React 等を使用し、洗練された一貫性のあるアイコンセットを採用。

### 2.3 ダッシュボード画面の新規導入 (Dashboard Page)
- **最近のシミュレーション履歴:** ユーザーが保存した最新のシミュレーション結果（3〜5件）をリストまたはカード形式で表示。
- **主要指標 (Market Summary):** S&P 500, 日経225, BTC などの主要資産の直近の騰落率を表示。
- **クイックアクション:** 「新規シミュレーション」「効率的フロンティア計算」などの主要機能へ直接遷移できる大きなボタン。

### 2.4 各既存画面のレイアウト調整
- **効率的フロンティア/積立シミュレーション:** 入力フォームとチャートを、それぞれカードとして整理し、レスポンシブなグリッドレイアウトで配置。

## 3. 非機能要件 (Non-Functional Requirements)
- **デザインの一貫性:** shadcn/ui のコンポーネントを積極的に活用し、一貫したルック＆フィールを維持。
- **パフォーマンス:** 装飾的なアニメーションを排除することで、高速な画面遷移とレンダリングを実現。

## 4. 受入基準 (Acceptance Criteria)
- ログイン後のトップ画面として、指定された3つの要素（履歴、指標、アクション）を含むダッシュボードが表示されること。
- サイドバーの折りたたみ/展開がスムーズに動作すること。
- すべての主要画面がカードベースのクリーンなデザインに更新されていること。
- 既存のシミュレーション機能が損なわれることなく動作すること。

## 5. 対象外 (Out of Scope)
- 派手なページ遷移アニメーションや、グラフ描画時のモーショングラフィックスの実装。
- モバイルアプリ（Native）対応。
