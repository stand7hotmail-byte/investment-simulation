# Initial Concept

etfや投資信託のリスク・リターンマップ、効率的フロンティア、自分で作成したポートフォリオの積み立て投資シミュレーション、リスクパリティ戦略、ポートフォリオのリバランス助言など投資に関する内容のwebサイトを作成したい。また、そのサイトからアフィリエイト等の収入を得たい。

## Target Users

このWebサイトは、投資初心者から中級レベルの個人投資家を主なターゲットとしています。投資に関する基本的な知識を学びたい方や、自身のポートフォリオをより深く分析し、最適化したいと考えている方に役立つ情報とツールを提供します。

## Goals

このWebサイトの最も重要な目標は以下の2点です。

1.  **ポートフォリオ作成・シミュレーションのためのインタラクティブなツールを提供する:** ユーザーが自身の投資戦略を具体的に検討し、その効果を視覚的に理解できるような、実践的なツール群を提供します。
2.  **アフィリエイトリンクや広告を通じて収益を生み出す:** サイトの運営を継続的に支え、より質の高いコンテンツとツールを提供するために、アフィリエイトプログラムや広告を通じて収益を確保します。

## Key Features

以下の重要な機能をWebサイトに含めます。

1.  **ETFと投資信託のリスク・リターンマップ:** さまざまなETFや投資信託のリスクとリターンの関係を視覚的に分かりやすく表示し、ユーザーが自身の投資目標に合った商品を選定する手助けをします。
2.  **効率的フロンティア計算・表示機能:** ユーザーが選択した複数の金融商品（銘柄）を基に、リスクを最小限に抑えつつリターンを最大化するポートフォリオの組み合わせ（効率的フロンティア）を計算し、グラフで表示します。
3.  **積み立て投資シミュレーション機能:** ユーザーが設定した積み立て額や期間、ポートフォリオの構成に基づいて、将来の資産形成を予測するシミュレーション機能を提供します。

## Monetization Strategy

アフィリエイト収入の主な源泉として、証券会社の口座開設を促進します。ユーザーが投資ツールを利用する中で、具体的な行動に移すための導線として、信頼できる証券会社の情報提供と口座開設への連携を行います。
